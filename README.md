# ✈️ Travel Booking Smart Contract (Soroban)

## 📌 Project Description
Travel Booking is a decentralized smart contract built on Stellar Soroban that allows users to securely create and manage travel bookings on-chain. It ensures transparency, ownership, and trust without relying on centralized booking systems.

## 🚀 What it does
This smart contract enables users to:
- Create travel bookings with destination and date
- Store booking data securely on the blockchain
- Mark bookings as paid
- Retrieve booking details anytime

It acts as a minimal backend for a decentralized travel booking platform.

## ✨ Features
- 🧾 On-chain booking records
- 🔐 User authentication using wallet address
- 💳 Payment status tracking
- 📦 Lightweight and gas-efficient design
- 🔎 Easy retrieval of booking data
- ⚡ Built using Soroban SDK (Rust)

## 🛠 Tech Stack
- Soroban SDK (Rust)
- Stellar Blockchain

## 📂 Functions Overview

### 1. create_booking
Creates a new travel booking.
- Inputs: user, destination, date
- Output: booking ID

### 2. pay_booking
Marks a booking as paid.
- Inputs: user, booking_id

### 3. get_booking
Fetches booking details.
- Inputs: booking_id
- Output: Booking struct
